Code Converter (Java ↔ Python)

This full-stack web application leverages Artificial Intelligence to seamlessly convert code snippets between Java and Python. The architecture is designed for efficiency and scalability.

Key Features & Technologies:
  • Backend: Built with Java Spring Boot, providing robust RESTful APIs.
  • AI Microservice: A Python Flask microservice handles interactions with Google's Gemini API for the core code conversion logic.
  • Frontend: Developed using Next.js, ensuring a responsive and modern user interface.
  • Developed a full-stack web app to convert code between Java and Python using AI.
  • Utilized Spring Boot for the main backend and a Flask microservice for Gemini API calls.
  • Implemented a responsive UI with Next.js.
Challenges & Learnings:

Integrating the Flask microservice with the Spring Boot backend while ensuring smooth data flow was a key challenge. This project deepened my understanding of microservice architecture, API design, and working with external AI services.
